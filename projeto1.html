Olá, estou estudando CSS

O Css é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Ele é usado para controlar a aparência e o layout de páginas web, permitindo que os desenvolvedores criem designs atraentes e responsivos. Com o CSS, é possível definir cores, fontes, espaçamento, posicionamento e muito mais, tornando as páginas web mais bonitas e funcionais.

Hoje na aula de CSS, aprendemos que existem 3 formas de de aplicar o CSS: inline, interno e externo.Vou usar os 3 metodos para praticar e entender melhor como cada um funciona.

Conderando isso tem algumas coisas importantes a serem lembradas, temos o seletor de ID que nos permite selecionar um elemento específico e aplicar estilos a ele, o Seletor de Classe que nos permite selecionar um grupo de elementos e aplicar estilos a eles, e o Seletor de Elemento que nos permite selecionar todos os elementos.

Palavras chaves:

  • Propriedades: Elementos que definem o estilo de um elemento HTML.
  • Valores: Valores que podem ser atribuídos a propriedades do CSS.
  • Atributo: Um valor que pode ser atribuído a uma propriedade do CSS.
  • Declaracao: Uma instrução que define um estilo para um elemento HTML.
  • Seletor: Elemento que permite selecionar um ou mais elementos HTML para aplicar estilos.
  • Frameworks: Softwares que facilitam o trabalho em algum programa.
  • Case-sensitive: Diferenciar letras maiúsculas e minúsculas
